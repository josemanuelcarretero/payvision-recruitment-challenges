Installation
------------

Download with `git`:

``` bash
$ git clone https://github.com/josemanuelcarretero/payvision-recruitment-challenges.git
```

Install dependencies with `npm`:

``` bash
$ npm install
```

Tests
------------

Run with `npm`:

``` bash
$ npm run test
```
